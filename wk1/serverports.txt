1. What is the definition of server port?

	A port is a virtual endpoint used in network communications. Ports are exposed, making them vulnerable and an important place to focus in cyber securit. 

2. What is server port range?
	There is a wide range of port numbers that can be / are used to make connections. These can be broken down into different categories by the IANA, each with their own 'port range'.
	- Well known:      ( 0 - 1023 )
	- Registered:   ( 1024 - 49151 )
	- Private:     ( 49152 - 65535 )
 

3. How many bits does number above represent?
	A server port number represents a 16 bit integer.
